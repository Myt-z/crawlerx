## CLI 命令详解

### 1. `web` — 网页爬取

**单页爬取：**

```bash
python main.py web "https://example.com/articles" \
    --rules "title:h1.title:text,date:span.date:text,link:a.read-more:attr:href"
```

`--rules` 格式：`字段名:CSS选择器:提取内容`，多个规则用逗号分隔。

| 提取内容 | 说明 | 示例 |
|----------|------|------|
| `text` | 提取元素内文本 | `title:h1:text` |
| `attr:href` | 提取指定属性值 | `link:a:attr:href` |
| `html` | 提取 innerHTML | `content:div.article:html` |
| 末尾加 `:multi` | 提取多个值（返回列表） | `tags:span.tag:text:multi` |

**翻页爬取：**

```bash
python main.py web "https://example.com/articles?page=1" \
    --rules "title:h2.title:text,author:span.author:text" \
    --paginate \
    --next-selector "a.next-page" \
    --max-pages 10 \
    --format csv
```

`--next-selector` 指定"下一页"按钮的 CSS 选择器，程序自动跟到最后一页。

**深度爬取（列表页 → 详情页）：**

```bash
python main.py web "https://example.com/articles" \
    --rules "title:h2:text,date:span.date:text" \
    --deep \
    --link-selector "a.title-link" \
    --detail-rules "content:div.article-body:html,author:span.author-name:text" \
    --max-items 50
```

先爬列表页提取标题和日期，再逐个进入详情页爬取正文。

**所有参数：**

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `url` | 目标网址（必填） | — |
| `--rules`, `-r` | 提取规则（必填） | — |
| `--paginate`, `-p` | 启用翻页 | 关闭 |
| `--next-selector` | 下一页按钮选择器 | `a.next` |
| `--deep`, `-d` | 深度爬取模式 | 关闭 |
| `--link-selector` | 详情链接选择器 | — |
| `--detail-rules` | 详情页提取规则 | — |
| `--max-pages` | 最多翻页数（0=不限） | 0 |
| `--max-items` | 最多爬取条目（0=不限） | 0 |
| `--format`, `-f` | 输出格式：json/csv/db | json |
| `--output`, `-o` | 输出文件路径 | data/crawl_output.json |
| `--table` | 数据库表名（format=db时） | crawl_data |
| `--concurrency`, `-c` | 并发请求数 | 10 |
| `--delay` | 请求间隔（秒） | 1.5 |

### 2. `video` — 视频下载

**下载 m3u8 流媒体：**

```bash
python main.py video "https://example.com/video/index.m3u8" --output my_video
```

流程：解析 m3u8 → 选最高画质 → 并发下载 ts 分片 → ffmpeg 合并为 mp4

**从网页中提取视频链接：**

```bash
python main.py video "https://example.com/page" --extract
```

自动扫描 `<video>` 标签、`<source>` 标签、js 中嵌入的 `.m3u8` 链接。

**直链视频下载：**

```bash
python main.py video "https://example.com/video.mp4" --output movie
```

支持断点续传、mp4/webm/flv 等格式。

**前置要求：** 下载 m3u8 视频需要安装 [ffmpeg](https://ffmpeg.org/download.html) 并加入系统 PATH。

### 3. `download` — 媒体批量下载

**下载图片：**

```bash
python main.py download "https://example.com/1.jpg,https://example.com/2.png" \
    --type image \
    --output-dir "data/my_images"
```

URL 用逗号分隔，自动识别扩展名。

**从网页提取并下载所有图片：**

```bash
python main.py download "https://example.com/gallery" \
    --page \
    --selector "img.photo" \
    --output-dir "data/photos"
```

**参数说明：**

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `url` | 文件URL（多个逗号分隔）或网页URL | — |
| `--type`, `-t` | image / file | file |
| `--page`, `-p` | 从网页提取媒体链接 | 关闭 |
| `--selector`, `-s` | CSS 选择器 | img |
| `--output-dir`, `-o` | 下载目录 | data/downloads |
| `--concurrency`, `-c` | 并发下载数 | 8 |

### 4. `analyze` — 数据分析

**查看数据概览：**

```bash
python main.py analyze data/demo/quotes.json --verbose
```

输出行数、列数、缺失值、描述性统计。

**某列的频率分布：**

```bash
python main.py analyze data/demo/quotes.json --column author --top-n 15
```

**词频统计（中文自动分词）：**

```bash
python main.py analyze data/articles.json --word-freq content --top-n 50
```

中文文本自动调用 jieba 分词，英文按空格切分。

**生成图表：**

```bash
# 柱状图
python main.py analyze data/demo/quotes.json --column author --chart bar

# 饼图
python main.py analyze data/demo/quotes.json --column author --chart pie --top-n 10

# 词云
python main.py analyze data/demo/quotes.json --word-freq text --chart wordcloud --output data/charts/my_wordcloud.png
```

**数据清洗 + 导出：**

```bash
python main.py analyze data/raw.json --clean --export data/clean.csv
python main.py analyze data/raw.json --clean --save-clean data/clean.json
```

**支持的输入格式：** `.json` / `.csv` / `.db`（SQLite）

## 配置调优

所有可调参数都在 [config.py](config.py) 中：

```python
# 爬虫行为
DEFAULT_DELAY = 1.5            # 请求间隔，被封就调大
DEFAULT_TIMEOUT = 20           # 请求超时（秒）
MAX_RETRIES = 3                # 失败重试次数
MAX_CONCURRENT = 10            # 异步并发数（网络好可调大）
USER_AGENT_ROTATION = True     # UA轮换（建议开启）

# TLS 指纹反检测
USE_CURL_CFFI = True            # 启用 curl_cffi 模拟 Chrome TLS 指纹
IMPERSONATE_TARGET = "chrome124" # 模拟目标浏览器

# 代理
PROXY_LIST = [
    # "http://127.0.0.1:7890",  # 填上就能用
]

# 断点续爬
CHECKPOINT_ENABLED = True       # 是否启用断点续爬
CHECKPOINT_INTERVAL = 3         # 每隔 N 页保存一次检查点

# 自适应重定位
ADAPTIVE_ENABLED = True         # 选择器失效时自动回退

# 视频
VIDEO_CHUNK_SIZE = 1024 * 1024  # 1MB 分片
VIDEO_MAX_WORKERS = 8           # ts 下载并发数
FFMPEG_PATH = "ffmpeg"          # ffmpeg 路径

# 存储路径
DB_PATH = data/db/crawler.db
```

## Python API 调用

除了命令行，也可以直接在 Python 代码中调用各模块：

```python
import asyncio
from crawlers import WebCrawler, VideoCrawler
from analysis import DataAnalyzer, Visualizer
from storage import JSONStorage, DBStorage

async def main():
    # 网页爬取
    async with WebCrawler(delay=1.0) as crawler:
        data = await crawler.crawl_paginated(
            start_url="http://quotes.toscrape.com",
            rules={
                "text":   {"selector": "div.quote span.text", "attr": "text"},
                "author": {"selector": "div.quote small.author", "attr": "text"},
                "tags":   {"selector": "div.quote a.tag", "attr": "text", "multiple": True},
            },
            next_selector="li.next a",
            max_pages=3,
        )

    # 保存
    JSONStorage("output.json").save(data)

    # 分析
    analyzer = DataAnalyzer(data)
    print(analyzer.value_counts("author", top_n=10))

    # 图表
    Visualizer.from_value_counts(
        analyzer.df, "author", chart_type="bar",
        title="Author Distribution", save_path="chart.png",
    )

    # 视频下载
    async with VideoCrawler() as vc:
        path = await vc.download_m3u8("https://example.com/video.m3u8", "my_video")

asyncio.run(main())
```

## 高级特性（v2.1 新增）

### TLS 指纹反检测

爬虫默认使用 `curl_cffi` 库模拟 **Chrome 124** 的 TLS 握手指纹，大幅降低被反爬系统识别的概率。

- **自动回退**：如果 `curl_cffi` 未安装，自动退回到 `httpx`，不影响任何功能
- **可切换目标**：在 `config.py` 中修改 `IMPERSONATE_TARGET` 可模拟其他浏览器（Firefox、Safari、Edge）
- **可关闭**：设置 `USE_CURL_CFFI = False` 则只使用 httpx

```python
# config.py
IMPERSONATE_TARGET = "chrome124"   # 可选: chrome123, firefox125, safari17_0, edge124
USE_CURL_CFFI = True               # False = 关闭 TLS 伪装
```

### 断点续爬

长时间翻页任务中断后可以从上次位置继续：

```python
# Python API 方式（传 checkpoint_key 参数即可）
data = await crawler.crawl_paginated(
    start_url="...",
    rules={...},
    next_selector="a.next",
    checkpoint_key="my_task",   # ← 给任务起个名
)
```

- 每 3 页（可配置）自动保存检查点到 `data/checkpoints/`
- Ctrl+C 中断后，同一 `checkpoint_key` 重新运行即可恢复
- 爬取完成后自动清理检查点文件
- 视频下载也会跳过已下载的 ts 分片，中断后可续传

### 自适应元素重定位

当网站改版导致 CSS 选择器失效时，自动尝试回退：

- **前缀回退**：如果 `div.product-list div.item` 找不到，自动尝试 `div.product-list`
- **指纹记忆**：成功的选择器保存到 `data/fingerprints/`，下次直接加载已验证的选择器
- **零配置**：默认启用，无需任何额外操作

```python
# config.py
ADAPTIVE_ENABLED = True              # False = 关闭自适应
ADAPTIVE_SIMILARITY_THRESHOLD = 0.6  # 相似度阈值
```

### URL 去重

内置去重机制，防止重复爬取：

- 翻页时检测到重复 URL 自动停止（防止无限循环）
- 深度爬取时自动去除重复的详情链接
- 视频下载时自动去除 m3u8 播放列表中重复的 ts 分片

## 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| `ImportError: No module named 'xxx'` | 没装依赖 | `pip install -r requirements.txt` |
| `ConnectionError` / 超时 | 网络问题或网站不可用 | 检查网络，调大 `DELAY`，添加代理 |
| 数据为空 | 网站结构变了 | F12 查看网页，更新 CSS 选择器 |
| 被网站封 IP | 请求太快 | 调大 `--delay`，加代理 |
| 中文乱码（图表） | 系统中文字体缺失 | 安装字体或将图表标题改为英文 |
| `ffmpeg` 未找到 | 没装 ffmpeg | [下载 ffmpeg](https://ffmpeg.org/download.html) 并加入 PATH |
| m3u8 合并失败 | ts 分片下载不全 | 调大 `VIDEO_MAX_WORKERS`，检查网络 |
| 词云中文乱码 | 中文字体未检测到 | 在 `visualize.py` 中指定字体路径 |
